# drf-events-api
[Events tracker REST API](https://77a230d26afe.ngrok.io/)

## Install Requirements
- Docker
- docker-compose

## Usage

```bash
git clone https://github.com/agustin-jimenez/drf-events-api
cd drf-events-api
docker-compose up
```
